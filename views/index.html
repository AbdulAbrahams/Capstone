Route Description

Method Path Description
login /login Post: A user
createUser /register Post: A user
fetchUsers /users get: Gets All the Users
fetchUser /user/:id get: Gets a Single User
updateUser /user/:id put: Updates a User
deleteUser /user/:id delete: Deletes a User

Product Route Description

Method Path Description
fetchProducts /products get: Gets all the Products
fetchProduct /product/:id get: A Product
addProduct /product post: Adds a Product
updateProduct /product/:id put: Updates a Product
deleteProduct /product/:id delete: Deletes a Product

Cart Route Description

Method Path Description
fetchCart /users/:id/cart get: Gets Users Cart
addToCart /users/:id/cart post: Adds a Product to the Cart
deleteFromCart /users/:id/cart/:cartID delete: Deletes a Product from the Cart
